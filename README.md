# Portfolio
M1|FA1 Portfolio Task by Faeldonia, Myca Joanne

## Live Demo

[https://emjaycodesss.github.io/portfolio/](https://emjaycodesss.github.io/portfolio/)

## Features

- Responsive design (desktop, tablet, mobile)
- Dark mode toggle with system preference detection
- Smooth GSAP animations and scroll effects
- Interactive project showcase with video previews
- Custom cursor and smooth scrolling
- Contact form integration

## Tech Stack

- HTML5, CSS3, JavaScript (ES6+)
- GSAP (animations)
- Lenis (smooth scrolling)
- Formspree (contact form)

## Getting Started

1. Clone the repository
   ```bash
   git clone https://github.com/emjaycodesss/portfolio.git
   cd portfolio
   ```

2. Run a local server
   ```bash
   python3 -m http.server 8000
   # or
   npx http-server -p 8000
   ```
